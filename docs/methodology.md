# Metodología — The NewsBreakers

## Objetivo

Detectar **desinformación** sobre **salud animal y brotes** a escala mundial, con soporte **multilingüe**.

## Fases

1. **MVP** — Página consultoría + API + keywords + indicadores.
2. **Datathon** — GDELT, scraping, fact-checkers (IFCN).
3. **Extensión** — Misma API desde el navegador.
